# DeepRay AI

AI-powered chest X-ray analysis system for lung disease detection and automated medical report generation.

## Overview

DeepRay AI is a graduation project focused on assisting healthcare professionals through deep learning solutions for chest X-ray interpretation.

The system combines:

- Lung disease classification from chest X-ray images
- AI-assisted medical report generation
- Web-based interface for image upload and diagnosis workflow
- Explainable AI using Grad-CAM heatmaps

## Features

### Chest X-ray Classification
Detects and classifies:

- Normal
- COVID-19
- Pneumonia
- Tuberculosis

### Medical Report Generation
Generates structured radiology-style reports from X-ray image features using deep learning and NLP techniques.

### User Interface

- Upload chest X-ray images
- View AI diagnosis results
- Medical workflow oriented interface
